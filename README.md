# AutoBet - WinUI 3 Application

Современное приложение на C# с использованием .NET WinUI 3.

## Возможности

✅ **Выбор режима игры**: Переключение между BetBoom и MelBet  
✅ **Светлая и тёмная тема**: Современный дизайн с поддержкой двух тем  
✅ **Панель настроек**: Удобная боковая панель с настройками  
✅ **Закруглённые элементы**: Все элементы интерфейса имеют плавные закругления  

## Требования

- Windows 10 версии 1809 (build 17763) или выше
- .NET 8.0 SDK
- Visual Studio 2022 (с установленной нагрузкой "Разработка классических приложений .NET")

## Установка и запуск

1. Откройте проект в Visual Studio 2022
2. Восстановите NuGet пакеты
3. Нажмите F5 для запуска приложения

### Запуск через командную строку

```powershell
dotnet restore
dotnet build
dotnet run
```

## Использование

### Переключение режима игры
Используйте переключатель в верхней части окна для выбора между BetBoom и MelBet.

### Изменение темы
1. Нажмите на иконку настроек (⚙️) в правом верхнем углу
2. Переключите "Тёмная тема" для изменения темы приложения
