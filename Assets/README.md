# Assets Directory

Эта директория содержит графические ресурсы приложения.

## Необходимые файлы изображений:

- **Square44x44Logo.png** - Иконка приложения 44x44
- **Square150x150Logo.png** - Плитка приложения 150x150
- **Wide310x150Logo.png** - Широкая плитка 310x150
- **SplashScreen.png** - Заставка при запуске
- **StoreLogo.png** - Логотип для магазина

Вы можете создать эти изображения с помощью любого графического редактора.
Для тестирования приложение будет работать и без этих файлов.
