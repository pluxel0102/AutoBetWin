# AutoBet - Создание установщика

## Шаг 1: Установите Inno Setup

1. Скачайте Inno Setup отсюда: https://jrsoftware.org/isdl.php
2. Установите программу (выберите Unicode версию)

## Шаг 2: Откройте скрипт

1. Запустите Inno Setup Compiler
2. Откройте файл: `AutoBet_Setup.iss`

## Шаг 3: Скомпилируйте установщик

1. В Inno Setup нажмите: Build → Compile (или F9)
2. Дождитесь окончания компиляции
3. Готовый установщик появится в папке: `Installer\AutoBet_Setup.exe`

## Что будет в установщике:

✅ Все файлы приложения (AutoBet.exe + библиотеки)
✅ Ярлык в меню Пуск
✅ Опциональный ярлык на рабочем столе
✅ Программа удаления (Uninstaller)
✅ Автоматическая очистка настроек при удалении

## Системные требования установщика:

- Windows 10 версии 1809 (Build 17763) или новее
- Windows 11 (рекомендуется)
- 64-bit система
- ~64 МБ свободного места

## Где будет установлено приложение:

По умолчанию: `C:\Program Files\AutoBet\`
Настройки: `C:\Users\[Username]\AppData\Local\AutoBet\`

## После создания установщика:

Файл `AutoBet_Setup.exe` в папке `Installer\` - это ваш готовый установщик!
Можете распространять его пользователям.

## Размер установщика:

Примерно 64 МБ (все библиотеки включены, .NET Runtime не требуется)
